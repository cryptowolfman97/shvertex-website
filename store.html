Installation Guide

How to install the SHV Store APK natively on Android

1

Download the file

Tap the download button above. Your browser will save the SHV_Store.apk file to your device.

2

Open the APK

Tap the "Download Complete" notification, or open your phone's File Manager app and navigate to your Downloads folder to find the file.

3

Allow Unknown Sources

Because we distribute directly instead of through Google Play, Android will ask you for security permission.

  • Tap Settings on the security popup.
  • Toggle the switch to Allow from this source.
  • Tap your back button to return to the installation.
4

Install & Login

Tap Install. Once finished, open the SHV Store app and log in with your SH Vertex account (or register directly inside the app) to access your ecosystem!
